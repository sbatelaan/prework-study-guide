HTML

  • The head element contains information about the webpage.
  • The body element represents the visible contwent shown to the user
  • Style elements defines CSS styles associated with a site
  • Base defines base URL
  • Link connects and external resource
  • Meta defines meta data such as character set, description, keywords, author, and viewport
  • Script embeds client side scripts in the HTML doc

CSS

  • Inline CSS styles an element directly into HTML file.
  • Internal style sheets imbeds entire style sheet directly into HTML file.
  • External is the most common method. Allows developers to keep all rules in a seperate file. Use link element placed in head.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can combine two or more elements using a comma rather than repeating code.
  • Block value assigns each image to start on a new line
  • Link element creates the connection to the external file
  • REL attribute specifies the relationship between current and linked document resource.
  • href specifies location (URL)
  • Class attribute- add styles to certain styles to certain elements, but not all.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git branch: list your branches. * will appear next to the currently active branch
  • git branch [branch-name]: create a new branch at the current commit
  • git log: show the commit history for the active branch
  • git pull: fetch and merge any commits from the tracking remote branch
